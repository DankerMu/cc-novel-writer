# DR-017: 竞品分析

## Executive Summary

当前 AI 小说写作工具市场以 **Sudowrite** 和 **NovelAI** 为代表，但均未实现**多 agent 结构化长篇协作**。本产品的核心差异化在于：端到端多 agent 协作 + 状态管理 + 一致性保证，这在市场中尚无竞品。

## Research Question

1. 主要竞品的功能、定价和用户评价？
2. 本产品的差异化优势在哪里？
3. 市场空白和进入机会？

## Methodology

对比分析 5 个主流 AI 写作工具的功能矩阵、用户反馈和定价模型。

## Key Findings

### 1. 竞品详细分析

#### Sudowrite
- **定位**：AI 写作助手，面向小说作者
- **核心功能**：
  - Story Engine：基于大纲自动生成章节
  - Describe：场景描写增强
  - Brainstorm：情节建议
  - Rewrite：句子级改写
  - Shrink Ray：文本压缩
- **定价**：$19-$49/月（按 AI 字数限制分级）
- **优势**：
  - 界面友好，学习成本低
  - Story Engine 支持长篇创作
  - 多种 AI 辅助模式
- **劣势**：
  - 无结构化状态管理
  - 长篇一致性依赖用户手动维护
  - 角色管理功能薄弱
  - 仅支持英文
- **用户评价**：
  - 好评："比 ChatGPT 更懂小说写作"
  - 差评："长篇后角色行为不一致"、"Story Engine 容易跑偏"

#### NovelAI
- **定位**：AI 辅助创作平台（文本 + 图像）
- **核心功能**：
  - Text Generation：基于自研模型的文本续写
  - Image Generation：AI 插图生成
  - Lorebook：世界观知识库
  - Memory：上下文记忆系统
- **定价**：$10-$25/月（按 tokens 和图像量分级）
- **优势**：
  - 自研模型，针对创意写作微调
  - Lorebook 提供基础的世界观管理
  - 图文结合
  - 隐私保护（本地处理选项）
- **劣势**：
  - 无结构化创作流程
  - 无自动大纲/情节规划
  - Lorebook 手动维护，无自动更新
  - 模型能力弱于 Claude/GPT-4
- **用户评价**：
  - 好评："自由度高，不审查内容"
  - 差评："模型质量不如 ChatGPT"、"长篇管理困难"

#### ChatGPT / Claude 直接使用
- **定位**：通用 AI 助手
- **核心功能**：直接对话式写作
- **定价**：$20/月（Pro 版本）
- **优势**：
  - 模型能力最强
  - 灵活度高
  - 多语言支持
- **劣势**：
  - 无专用写作功能
  - 无状态管理
  - 对话上下文有限
  - 每次会话需重新建立上下文
- **用户评价**：
  - 好评："短篇写作质量高"
  - 差评："长篇无法维持一致性"、"每次都要重复背景"

#### Jasper / Copy.ai
- **定位**：商业文案写作
- **核心功能**：模板化内容生成
- **定价**：$39-$99/月
- **与本产品关系**：不直接竞争，面向商业写作

#### AI Dungeon
- **定位**：交互式故事体验
- **核心功能**：实时交互叙事
- **定价**：免费/$9.99/月
- **与本产品关系**：偏娱乐，不面向严肃创作

### 3. 差异化定位

**独特卖点（USP）**：

1. **端到端结构化创作**：从世界观到成稿的完整流水线
2. **自动一致性保证**：状态管理 + NER 检查 + 伏笔追踪
3. **多 agent 协作**：专业化分工（WorldBuilder、CharacterWeaver 等）
4. **中文原生支持**：面向中国网文市场

**竞争壁垒**：
- 基于 Claude Code 的 agent team 能力（竞品无法轻易复制）
- 结构化状态管理系统（积累数据优势）
- 针对中文长篇小说的专业化优化

### 4. 市场进入策略

**机会窗口**：
- Sudowrite/NovelAI 未进入中文市场
- 国内 AI 写作工具无长篇结构化能力
- 网文作者对效率工具需求迫切

**风险**：
- Claude Code 门槛高（需要命令行操作）
- 大厂可能快速跟进（如百度文心、阿里通义）
- 定价需要与 ChatGPT Pro（$20/月）竞争

### 5. 定价建议

| 层级 | 价格 | 功能 | 目标用户 |
|------|------|------|----------|
| 免费 | $0 | 3 章试用 | 所有 |
| 基础版 | $15/月 | 单部小说，基础 agent | 业余爱好者 |
| 专业版 | $29/月 | 多部小说，全部 agent，一致性检查 | 网文作者 |
| 团队版 | $49/月 | 多用户协作，API 接入 | 工作室 |

（注：以上不含 Claude API 成本，需要额外收费或包含在订阅中）

## Conclusion

**决策建议**：
1. **差异化明确**：多 agent 协作 + 一致性保证是独特卖点
2. **首要竞争对手**：Sudowrite（功能最接近），ChatGPT（用户最多）
3. **进入策略**：聚焦中文网文市场，避免与英文工具正面竞争
4. **关键劣势**：需要 Claude Code 环境，门槛高于 SaaS 产品
5. **后续考虑**：长期可能需要 Web UI 降低使用门槛
